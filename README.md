# 📝 Simple Vim Config

Super simple daily Vim config — no plugins, minimal, clean.
👉 Jadi workflow kamu:  
```bash
git clone https://github.com/osiic/.vim.git ~/.vim
cd ~/.vim
bash install.sh
vim
````

## 📦 Install

Clone repo langsung ke `~/.vim`:

```bash
git clone https://github.com/osiic/.vim.git ~/.vim
cd ~/.vim
bash install.sh
````

## 🔧 Update

```bash
cd ~/.vim
bash update.sh
```

## ♻️ Reinstall

```bash
cd ~/.vim
bash reinstall.sh
```

## ❌ Uninstall

```bash
cd ~/.vim
bash uninstall.sh
```

Kalau ada `.vimrc` lama, otomatis dibackup jadi `~/.vimrc.backup`.
